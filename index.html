📋 Setup (one time, ~5 min)
1. Create a new Google Sheet with these column headers in Row 1:
Company | Contact | Email | Phone | Industry | State | Headcount | Persona | LinkedIn | Track | Date Saved | Time
2. In your Sheet → Extensions → Apps Script
3. Paste this exact code (replaces everything):
function doGet(e) { return handleRequest(e); } function doPost(e) { return handleRequest(e); } function handleRequest(e){ var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheets()[0]; var p = (e.parameter && Object.keys(e.parameter).length) ? e.parameter : JSON.parse(e.postData.contents||'{}'); sh.appendRow([ p.company||'', p.contact||'', p.email||'', p.phone||'', p.industry||'', p.state||'', p.headcount||'', p.persona||'', p.linkedin||'', p.track||'', p.saved||'', p.time||'' ]); return ContentService .createTextOutput('ok') .setMimeType(ContentService.MimeType.TEXT); }
4. Click Deploy → New deployment → Web App
5. Set Execute as: Me · Who has access: Anyone
6. Copy the Web App URL and paste it above
⚠ If you already deployed: go to Deploy → Manage deployments → edit → create new version to pick up any script changes.
